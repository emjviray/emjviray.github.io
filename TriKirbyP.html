Project Description:

The TriKirby Index is a machine learning-driven pitch command metric developed to quantify a pitcher’s ability to consistently repeat release mechanics at the pitch-type level. Using TrackMan data from NCAA Division I baseball and UC San Diego fall scrimmages, the project models variability in vertical and horizontal release angle and location as indicators of command. These features are transformed into percentile-based consistency scores and combined into a single interpretable metric using Random Forest-derived feature importances, with Logistic Regression used as a baseline modeling approach. The resulting index enables cross-pitch and cross-player comparisons relative to NCAA benchmarks. This work provides actionable insights for coaching decisions, player development, and pitch design while also contributing to cognitive science research on learning and performance optimization in skilled athletes.
