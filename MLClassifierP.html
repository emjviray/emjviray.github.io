Project Description:

Conducted a systematic benchmark study comparing five supervised machine learning classifiers — Logistic Regression, Decision Trees, Random Forest, K-Nearest Neighbors, and Support Vector Machines — across four real-world UCI datasets (Heart Disease, Adult Income, Wine Quality, and Bank Marketing). This search project ended up getting an (A-) which was the highest grade recieved from the Fall 2025 COGS 118A Supervised Machine Learning Models Course. Each model was optimized using RandomizedSearchCV with 3-fold stratified cross-validation, then evaluated across three train/test splits (20/80, 50/50, 80/20) and three independent trials per configuration to ensure stable, reproducible results. Performance was measured using accuracy, F1 score, and classification error, with findings visualized through heatmaps, error curves, stability charts, and cross-validation scatterplots. Random Forest emerged as the strongest and most consistent classifier across all datasets and conditions, while Logistic Regression proved reliable on structured data, and SVM showed the most instability without full tuning. The study demonstrated that classifier performance is shaped by both model architecture and inherent dataset difficulty, providing practical insight into model selection for applied machine learning contexts.
