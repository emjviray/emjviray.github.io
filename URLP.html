Project Description:

Developed a machine learning pipeline to detect malicious URLs by analyzing their lexical and structural characteristics, without relying on webpage content or external databases. The project employed three complementary approaches: Logistic Regression as an interpretable baseline, XGBoost as a high-performance nonlinear classifier for feature importance analysis, and K-Means clustering for unsupervised pattern discovery. XGBoost achieved 93% accuracy and identified the top predictive signals — including URL entropy, path complexity, uppercase letter frequency, and the presence of suspicious keywords like "login" — which K-Means independently validated by naturally separating malicious from benign URLs with a purity score of 0.82. The project culminated in an interactive URL risk scoring demo that takes any URL as input and returns a real-time maliciousness probability score with an explanation of which features drove the prediction, translating the research findings into a practical, user-friendly cybersecurity tool.
